Chat history plugin
« on: March 08, 2007, 12:46:06 AM »
history plugin for HeXHub will record all chatter that goes in mainchat.
Users can use the !history trigger to get the last lines of mainchat.
The project is open source software licensed under GNU/GPL
and can be found at http://sourceforge.net/projects/hexhub
Direct link: http://sourceforge.net/project/showfiles.php?group_id=165882&package_id=202041&release_id=447600
DO NOT use versions prior to 0.3 those contains bugs and a memory leak.
