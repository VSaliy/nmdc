Hublist Registration Server
« on: September 19, 2007, 09:26:58 AM »
Hublist Registration Server is a plugin for hublist admins that can convert registration requests sent by hubs to "add hub" requests sent to hublist's web server. If you use this plugin you need to open port 2501 in hub (standard port for hublist registration servers).
The registration server has flood protection, doesn't flood the web server with requests, supports cookies and web redirects, can verify hubs, check for duplicates, use a pinger to detect fakes, etc.
