The script is now compatible for both playing in main and pm.
I''t has come to my notice that i have to clarify a few things: You now have the abbility to choose where you want to play by writing the !startdeal cmd either in pm or in main, that means that were ever you write the cmd the game will be played there.
!dealhelp         This help file (works in both pm and main)
!startdeal      Starts a new game (works in both pm and main)
!dealscores    Show top 10 High Scores (works in both pm and main)
!resetdeal      Reset the game ( this cmd is for main since it''s for OPs only, and you cannot reset your own game )
!resetscores   Reset the High Scores ( this cmd is for main since it''s OPs only )
