This is not really a client but a hublink program, and can be downloaded from here: http://downloads.sourceforge.net/hexhub/MultiChat10.zip.
Like any hublink program it has options for message formatting, forbid list, ignore list (for all hubs or for each hub in particular), flood detection, can save logs (full traffic logs or only chat), etc.
If flood is detected, instead of discarding messages it groups them into a bigger message.
