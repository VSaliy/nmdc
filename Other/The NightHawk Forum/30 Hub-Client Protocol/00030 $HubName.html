$HubName

Syntax
Quote$HubName <hubname>|
Description
This command is part of the Client-Hub Handshake and provides the name of the hub to a client.
â€¢ <hubname> is the name you use for the hub.
Extensions
It can be sent at any time, changing the name of the hub. This variable should only be used as a sort of description of the hub, a topic.
Client1 and Client2 in the same hub might have recieved different <hubname>, this can occur in nmdchub. So <hubname> should not be used in a way to uniquely identify hubs.
DC++ expects the ampersand, pipe, and dollar sign (&, |, $) to be escaped with their HTML equivalents.
