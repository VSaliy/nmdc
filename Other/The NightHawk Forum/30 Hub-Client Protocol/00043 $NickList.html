$NickList
« on: May 02, 2007, 09:14:53 PM »

Syntax
Quote$NickList <nick1>$$<nick2>$$<nick3>$$...|
Description
This command is used by a hub to provide a list of all logged in users to a client. The list is delimited by the "$$" token
