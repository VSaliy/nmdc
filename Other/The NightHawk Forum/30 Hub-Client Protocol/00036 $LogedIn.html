$LogedIn
« on: May 02, 2007, 09:23:22 PM »

Syntax
Quote$LogedIn <username>|
Description
This command is sent by the hub in response to a successful $GetPass/$MyPass login sequence. This is only sent to Ops.
Mind the spelling, this command is sent including the typo.
The current version of DC++ doesn''t use $LogedIn, and has not since (at least) version 0.181.
