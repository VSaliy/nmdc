$MultiSearch
« on: May 02, 2007, 09:18:59 PM »

Syntax
Active user
Quote$MultiSearch <ip>:<port> <searchstring>
Description
This command was as a client side command to take advantage of hub linking feature in the NeoModus Direct Connect Hub software. When a local client sends $MultiSearch to the hub, it is forwarded on to every hub which is linked to by UDP, as in the $Search hub<->hub protocol command.
This command is used in conjunction with $MultiConnectToMe. Neither is supported in DC++, though there was limited support in a couple versions of oDC. This support was problematic, see the description in $MultiConnectToMe.
For syntax of the parameters, see $Search.
