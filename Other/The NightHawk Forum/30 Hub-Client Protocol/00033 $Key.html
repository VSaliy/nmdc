$Key
« on: May 02, 2007, 09:26:24 PM »

Syntax
Quote$Key <calculated key>|
Description
The $Key command is issued in response to a $Lock command sent by the remote host. When used in client to client communication, or in hub to client communication, the $Key is calculated per the algorithm used in the example source code in LockToKey. The $Key is also used when a hub registers itself with a hub list server. In that case, the first character in the key is calculated differently.
