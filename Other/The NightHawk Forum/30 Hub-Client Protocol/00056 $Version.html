$Version
« on: May 02, 2007, 08:48:37 PM »
Syntax
Quote$Version <version>|
Description
This command should be sent after the hub has sent $Hello to client.
The last 1.x version of NMDC issued 1.0091 as a version number. When run on a computer with alternate decimal separators, it used the system default. Thus, DC++ issues 1,0091 as a version number, regardless of user locale.
Remember multiple separators can be used, the comma (,) and period (.) being the most common.
