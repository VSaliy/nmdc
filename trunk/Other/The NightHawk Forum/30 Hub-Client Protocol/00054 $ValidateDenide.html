$ValidateDenide
« on: May 02, 2007, 08:51:24 PM »

Syntax
Quote$ValidateDenide <nick>|
Description
This is part of the client-hub handshake. The hub will return this command to the client following $ValidateNick if the requested nick is already taken, or is a special reserved nick (such as "Hub-Security").
• <nick> is the nickname that was denied.
• After receiving this command the client generally displays the "Your nick was already taken" message (this message varies depending on the client).
• This spelling of this command is not a mistake.
