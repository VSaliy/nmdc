$OpList
« on: May 02, 2007, 09:11:58 PM »

Syntax
Quote$OpList <op1>$$<op2>$$<op3>$$...|
Description
This command is used by a hub to transmit a list of all operators to a client. This list is delimited by the "$$" token.
Note that $OpList is a subset of $NickList. If there are no connected operators, the server must send "$OpList".
The original NMDC v1 Hub also included all Registered Bots (scripts) in the $OpList. New hubs may do the same thing or provide the option of not showing the Bot (script) as an Op.
