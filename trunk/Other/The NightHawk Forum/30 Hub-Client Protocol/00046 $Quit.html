$Quit
« on: May 02, 2007, 09:10:21 PM »

Syntax
Quote$Quit <nick>|
Description
• This command is only sent by the hub. If you''re programming a client, do not send the hub a $Quit. GLO-Search made this mistake, which is a bug that has been fixed although no new version has been released since the bugfix.
• The command indicates that <nick> has disconnected from the hub.
Example
Quote$Quit BlackClaw|
