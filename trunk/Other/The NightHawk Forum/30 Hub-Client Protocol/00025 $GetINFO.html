$GetINFO
« on: May 02, 2007, 09:43:37 PM »

Syntax
Quote$GetINFO <othernick> <nick>|
Description
Request someones $MyINFO from the Hub.
• <nick> is this sending client''s nick.
• <othernick> is the nick of the user that <nick> wants to know about.
The server must respond with exactly the $MyINFO command sent by <othernick> to the hub.
