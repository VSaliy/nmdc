$GetNickList
« on: May 02, 2007, 09:42:51 PM »

Syntax
Quote$GetNickList|
Description
This command causes the hub to send two lists of logged-in users. The format is a $$-separated list of nicks.
Quote$NickList <nick1>$$<nick2>$$<nick3>$$...
$OpList <op1>$$<op2>$$<op3>$$...
Note that the $OpList list is a subset of the $NickList list. If there are no connected operators, the server must send just "$OpList".
