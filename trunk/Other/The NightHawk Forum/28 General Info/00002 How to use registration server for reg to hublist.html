VerliHub:
Quote!gc
Search for hublist_host variable
Copy all regservers if you don't want to get lost,
write "!= hublist_host "
And paste old regservers, press spacebar once then write there "reg.hublist.dy.fi"
and press enter.   
Ynhub:
QuoteGo to Settings -> Miscellaneous -> Hublist
For field Name use reg.hublist.dy.fi
Description, something like Regserver for hublist.dy.fi
To Host (DNS/IP) enter reg.hublist.dy.fi
To Port: enter 2501
Check Use this public registration server
Click Save as
Check Enable hublist registrations
